## 📋 完整部署流程

### 第一步：注册并配置（5分钟）

1. **注册账号**
   - 访问：https://dashboard.cpolar.com/signup
   - 邮箱注册（免费）

2. **获取 Authtoken**
   - 登录控制台
   - 复制 authtoken

3. **配置（运行一次）**
   ```bash
   cd /home/hongfuyang/公共/网站建设
   ./启动cpolar.sh
   # 或直接运行：
   cpolar authtoken YOUR_TOKEN
   ```

### 第二步：启动服务（1分钟）

```bash
# 方式 1：使用启动脚本（推荐）
./启动cpolar.sh

# 方式 2：直接命令
cpolar http 80
```

### 第三步：获取并分享地址

启动后会显示：
```
Forwarding    https://xxxx.cpolar.cn -> http://localhost:80
```

**这个地址就是公网访问地址！**
- 把地址发给任何人
- 他们直接访问即可
- 无需任何密码或配置

### 第四步：（可选）绑定域名 ndtool.cn

**免费版限制：**
- 地址会变化（每次重启）
- 格式：xxxx.cpolar.cn

**VIP 版（约 10元/月）：**
- 固定域名
- 可绑定 ndtool.cn
- 更多功能

## 💡 注意事项

### 电脑需要开机
- 本地电脑作为服务器
- 关机后网站无法访问
- 建议设置不自动休眠

### 免费版限制
- 随机域名（每次重启会变）
- 1个在线隧道
- 不限流量

### 升级 VIP 的好处
- 固定域名
- 自定义域名绑定（ndtool.cn）
- 多个隧道
- 更快的速度

## 🆘 常见问题

### Q: 访客真的不需要密码吗？
**A: 是的！访客直接访问网址即可，就像访问普通网站一样，无需任何密码、验证或配置。**

### Q: 安全吗？
A: 安全。Cpolar 使用 HTTPS 加密传输，只是转发流量，不存储内容。

### Q: 速度怎么样？
A: 很快！国内服务器，延迟通常在 20-50ms。

### Q: 免费版够用吗？
A: 测试和临时使用完全够用。如需固定域名，建议升级 VIP（10元/月）。

### Q: 可以绑定我的域名 ndtool.cn 吗？
A: 可以，但需要升级 VIP。免费版只能使用 xxxx.cpolar.cn 格式。
